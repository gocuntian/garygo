反射的作用是在程序运行时获取数据的类型信息和值信息。

golang提供了一个官方包reflect来获取运行时数据。关键点在于两个方法TypeOf和ValueOf。

TypeOf可以获取到类型信息，ValueOf可以获取值信息。
